Hello there, thanks for taking a look at my application! This repository includes all of the maps shown in the showcase video and a custom world gen add-on.

All of the worlds provided can be loaded in Java and evaluated that way, each .zip file includes a list of teleport commands/coordinates to make navigation easier.

As for the add-on, it will need to be loaded in Minecraft: Bedrock Edition, which can be done in one of two ways:

1. Double-click the .mcaddon file included in this file and/or the git main directory and it will automatically import it for use. Just attach it to a new world and you should be good to go!
2. Alternatively, clone the git repository, then copy the "Worldgen_Project_AC" folders from the Content directory and paste them into the development_behavior_packs and development_resource_packs folders
that are located in the com.mojang directory. Just make sure they go into their matching folder!

To find the com.mojang directory, press Win + R and enter this text:

%appdata%\Minecraft Bedrock\Users\Shared\games\com.mojang

A VS Code .workspace file is also included in the repository for convenience of viewing the code.

Here's also a list of commands that should make evaluating the Portfolio add-on a bit easier once the it is loaded!

Locating Biomes:

/locate biome ac:candy_fields
/locate biome ac:nether_plains

Locating Structures:

/locate structure ac:candy_fields_village
/locate structure ac:donut_dungeon

These are the primary things to be looking for. 

There's also small handful of extra stuff in the add-on pack that's at varying stages of completeness, but those have been disabled in-game for the sake of simplicity.
